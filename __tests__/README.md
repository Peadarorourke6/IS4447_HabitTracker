## About
A React Native habit tracking app built with Expo and Drizzle ORM. Users can create habits, log daily activity, set weekly targets, and view insights and streaks.

## Features
Create, edit and delete habits with categories and colours.
Log daily habit activity with optional notes.
Set weekly/monthly targets with progress tracking.
Insights screen with bar charts and streak tracking.
Search and filter habits by name and category.
Login, register, logout and delete profile.
Light/dark mode toggle.
Local SQLite storage via Drizzle ORM.

## Tech Stack
- React Native + Expo
- Expo Router 
- Drizzle ORM + SQLite
- AsyncStorage
- Jest (testing)
